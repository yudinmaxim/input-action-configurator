## 🔧 Этап 1: Подготовка

### 1.1 Установка зависимостей
```bash
npm install vue-i18n@9
```

### 1.2 Настройка i18n в приложении
- Создать `src/shared/i18n/index.ts`
- Создать `src/shared/i18n/locales/ru.json`
- Создать `src/shared/i18n/locales/en.json`
- Интегрировать в `src/main.ts`
