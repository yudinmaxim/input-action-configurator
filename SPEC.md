# Input Action Configurator - Техническое Задание

## 1. Общее Описание

**Название проекта:** Input Action Configurator
**Тип:** Desktop приложение (Tauri + Vue 3)
**Назначение:** Визуальный редактор конфигурационных файлов для InputActions плагинов (kwin/standalone)

**Целевая платформа:** Linux desktop (KDE Plasma с KWin)

**Конфигурационный файл:** `~/.config/inputactions/config.yaml`

---

## 2. Функциональные Требования

### 2.1 Загрузка и Сохранение Конфигурации

- Чтение YAML конфигурации из `~/.config/inputactions/config.yaml`
- Автоматическое определение изменений в файле (file watching)
- Сохранение конфигурации с форматированием
- Валидация YAML перед сохранением

### 2.2 Управление Устройствами

Поддерживаемые устройства (из документации):
- **keyboard** - клавиатура
- **mouse** - мышь (включая trackpoints)
- **touchpad** - тачпад
- **touchscreen** - сенсорный экран

Для каждого устройства доступны:
- Свойства (Properties)
- Правила устройств (Device Rules)
- Список триггеров

### 2.3 Создание и Редактирование Триггеров

Типы триггеров по устройствам:

**Keyboard:**
- `shortcut` - комбинация клавиш

**Mouse:**
- `circle` - круговое движение
- `press` - нажатие кнопки
- `stroke` - жест (движение с изменением направления)
- `swipe` - свайп (прямое движение)
- `wheel` - вращение колеса

**Touchpad:**
- `circle` - круговое движение (1-4 пальца)
- `click` - клик
- `hold` - удержание (1-4 пальца)
- `pinch` - щипок (масштабирование)
- `rotate` - вращение (2-4 пальца)
- `stroke` - жест (1-4 пальца)
- `swipe` - свайп (1-4 пальца)
- `tap` - тап

**Touchscreen:**
- `circle` - круговое движение
- `hold` - удержание
- `pinch` - щипок
- `rotate` - вращение
- `stroke` - жест
- `swipe` - свайп
- `tap` - тап

### 2.4 Создание Действий

Типы действий:
- `input` - симуляция ввода (клавиатура/мышь)
- `command` - выполнение команды
- `plasma-shortcut` - Plasma горячая клавиша
- `replace-text` - замена текста
- `sleep` - задержка

## 4. Справочник: Триггеры

### 4.1 Общие Свойства Триггеров

| Свойство | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `type` | enum | Тип триггера | (обязательно) |
| `accelerated` | bool | Использовать ускоренный delta | false |
| `actions` | list | Список действий | (обязательно) |
| `block_events` | bool | Блокировать входные события | true |
| `clear_modifiers` | bool | Очистить модификаторы | true* |
| `conditions` | list | Условия активации | - |
| `end_conditions` | list | Условия завершения | - |
| `id` | string | Уникальный ID | - |
| `resume_timeout` | time | Время для повторного выполнения | 0 |
| `set_last_trigger` | bool | Устанавливать last_trigger переменные | true |
| `threshold` | float/range | Порог активации | - |
| `speed` | enum | Скорость (fast/slow) | - |

*`true` если есть input действие

### 4.2 Жизненный Цикл Триггера

- `begin` - Начало выполнения
- `update` - Обновление (проверка условий направления)
- `tick` - Тик (интервал как у time-based триггеров)
- `end` - Завершение (выполнено)
- `cancel` - Отмена

### 4.3 Типы Триггеров

#### Swipe Триггер

**Свойства:**

| Свойство | Тип | Описание |
|----------|-----|----------|
| `angle` | range | Диапазон углов (a-b, где 0-360) |
| `direction` | enum | Направление |
| `bidirectional` | bool | Разрешить движение в обратную сторону |
| `lock_pointer` | bool | Зафиксировать позицию курсора |

**Направления:**
- `left`, `right`, `up`, `down`
- `left_right`, `up_down` (двунаправленные)
- `left_up`, `left_down`, `right_up`, `right_down` (диагональные)
- `left_up_right_down`, `left_down_right_up` (двунаправленные диагональные)
- `any`

**Диапазоны углов:**
- 0° - right, 90° - up, 180° - left, 270° - down
- Формат: `a-b` (например `30-330`)
- Если `a > b`, диапазон "оборачивается" через 0

#### Circle Триггер

**Свойства:**

| Свойство | Тип | Описание |
|----------|-----|----------|
| `direction` | enum | Направление вращения |
| `lock_pointer` | bool | Зафиксировать позицию курсора |

**Направления:**
- `clockwise`
- `counterclockwise`
- `any`

#### Hold Триггер

Временной триггер. Выполняется удержанием пальцев.

#### Tap Триггер

Выполняется касанием.

#### Click Триггер (touchpad)

Выполняется касанием тачпада с нажатием.

#### Press Триггер (mouse)

**Свойства:**

| Свойство | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `instant` | bool | Начинать немедленно | false |

#### Stroke Триггер

Жест с изменением направления.

#### Pinch Триггер

Щипок (масштабирование).

#### Rotate Триггер

Вращение (2-4 пальца).

#### Wheel Триггер

Вращение колеса мыши.

#### Shortcut Триггер (keyboard)

Комбинация клавиш.

## 5. Справочник: Действия (Actions)

### 5.1 Общие Свойства Действий

| Свойство | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `conditions` | list | Условия выполнения | - |
| `conflicting` | bool | Активировать conflict resolution | true |
| `interval` | float/str | Интервал выполнения | 0 |
| `limit` | uint | Лимит выполнений (0 = без лимита) | 0 |
| `on` | enum | Событие жизненного цикла | `end` |
| `threshold` | float/range | Порог выполнения | - |

**Интервал:**
- `0` - один раз за событие
- `+` - только при положительном delta
- `-` - только при отрицательном delta
- `число` - при достижении суммарного delta

**События (`on`):**
- `begin` - начало
- `cancel` - отмена
- `end` - завершение
- `end_cancel` - завершение при отмене
- `tick` - тик
- `update` - обновление

### 5.2 Input Действие (input)

Симуляция ввода клавиатуры и мыши.

**Свойства:**

| Свойство | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `input` | list | Список действий ввода | (обязательно) |
| `delay` | time | Задержка между действиями | 0 |

#### KeyboardAction

| Формат | Описание |
|--------|-----------|
| `+key` | Нажать клавишу |
| `-key` | Отпустить клавишу |
| `key1+key2+...` | Нажать несколько клавиш |
| `text: [текст]` | Ввести текст |

#### MouseAction

| Формат | Описание |
|--------|-----------|
| `+button` | Нажать кнопку |
| `-button` | Отпустить кнопку |
| `button1+button2+...` | Несколько кнопок |
| `move_by [x] [y]` | Сдвинуть курсор |
| `move_by_delta [множитель]` | Сдвинуть на delta |
| `move_to [x] [y]` | Переместить к курсор |
| `wheel [x] [y]` | Вращать колесо |

### 5.3 Command Действие

Выполнение shell команды.

**Свойства:**

| Свойство | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `command` | string | Команда для выполнения | (обязательно) |
| `async` | bool | Выполнить асинхронно | false |

### 5.4 Plasma Shortcut Действие

Отправка Plasma горячей клавиши.

**Свойства:**

| Свойство | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `shortcut` | string | Например `Alt+F4` | (обязательно) |

### 5.5 Replace Text Действие

Замена текста.

**Свойства:**

| Свойство | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `replace_with` | string | Текст для вставки | (обязательно) |

### 5.6 Sleep Действие

Задержка.

**Свойства:**

| Свойство | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `time` | time | Время задержки | (обязательно) |

## 8. UI/UX Требования

### 8.1 Основной Интерфейс

1. **Список устройств** - боковая панель с деревом устройств
2. **Панель триггеров** - основная область для выбранного устройства
3. **Редактор свойств** - форма для редактирования свойств триггера/действия
4. **Превью YAML** - отображение текущего конфига

### 8.2 Формы Редактирования

Для каждого типа триггера создается форма с:
- Выбором типа устройства
- Выбором типа триггера
- Настройкой направления/угла
- Настройкой количества пальцев
- Настройкой порогов
- Списком действий
- Условиями активации

### 8.3 Валидация

- Проверка уникальности ID
- Проверка допустимых значений
- Проверка совместимости триггеров
